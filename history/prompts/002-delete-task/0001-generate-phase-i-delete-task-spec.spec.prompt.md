## Prompt

# Feature Specification: Delete Task

## Overview
Implement a function to delete an existing task from the in-memory Todo list by ID. This is the second basic level feature for Phase I, enabling users to remove tasks.

## Data Model
Reference: specs/data_model.md (shared across features)
- Class: Task
  - id: int (unique identifier for deletion)
  - title: str
  - description: str
  - completed: bool

The in-memory storage is a list[Task] in the main app module, shared with other features.

## Requirements
- Command: 'delete <id>' (CLI input parsing, id as integer)
- Validate ID exists in list; if not, handle gracefully
- Remove task from list
- Output confirmation: "Task deleted: ID {id} - {title}"
- Integrate with existing list (remove by index or ID match)

## Quality Gates (per Constitution Principle 5)
- Type-safe with full hints (e.g., def delete_task(tasks: list[Task], task_id: int) -> None)
- Error handling: Raise ValueError if ID not found, print user-friendly message (e.g., "Task ID not found")
- Clean code: PEP 8, efficient removal (O(n) ok for small list), docstring for function
- Testable: Function modifies list in place but is idempotent if ID missing
